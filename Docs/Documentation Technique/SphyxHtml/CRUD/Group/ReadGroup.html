Read Group¶

Ce module permet de récupérer des informations sur les groupes, les utilisateurs associés à des groupes, et les tâches assignées à des groupes via une API PHP.

Fonctions¶

get_data(table, columns='*', filter_column=None, filter_value=None, join_table=None, join_condition=None)¶

Récupère des données d’une table spécifique via l’API.

Paramètres:

  • table (str) – Nom de la table principale à interroger.

  • columns (str, optionnel) – Colonnes à récupérer (par défaut toutes les colonnes).

  • filter_column (str, optionnel) – Colonne pour appliquer un filtre.

  • filter_value (str, optionnel) – Valeur du filtre pour la colonne spécifiée.

  • join_table (str, optionnel) – Nom de la table pour effectuer une jointure.

  • join_condition (str, optionnel) – Condition de jointure entre les tables.

Renvoie:

Les données récupérées sous forme de dictionnaire JSON ou un message d’erreur.

Type renvoyé:

dict ou str

Exemple d’utilisation :

data = get_data("Group", columns="id_group, name", filter_column="id_group", filter_value=1)
get_group(id_group=None, name=None)¶

Récupère les informations d’un ou plusieurs groupes. Si aucun paramètre n’est spécifié, retourne tous les groupes.

Paramètres:

  • id_group (int, optionnel) – Identifiant du groupe.

  • name (str, optionnel) – Nom du groupe.

Renvoie:

Une liste de dictionnaires contenant les informations des groupes ou un message indiquant qu’aucune correspondance n’a été trouvée.

Type renvoyé:

list ou str

Exemple d’utilisation :

groups = get_group(id_group=1)
all_groups = get_group()
get_group_user(id_group)¶

Récupère une liste des utilisateurs appartenant à un groupe donné.

Paramètres:

id_group (int) – Identifiant du groupe.

Renvoie:

Une liste des identifiants des utilisateurs associés au groupe.

Type renvoyé:

list

Exemple d’utilisation :

users = get_group_user(1)
get_group_task(id_group)¶

Récupère une liste des tâches assignées à un groupe donné.

Paramètres:

id_group (int) – Identifiant du groupe.

Renvoie:

Une liste des identifiants des tâches associées au groupe.

Type renvoyé:

list

Exemple d’utilisation :

tasks = get_group_task(1)

Variables Globales¶

url¶

URL de l’API PHP utilisée pour les requêtes.

Type:

str

Valeur : "https://kicekifeqoa.alwaysdata.net/api.php"

Exemple Complet¶

# Récupérer tous les groupes
all_groups = get_group()

# Récupérer un groupe par son identifiant
group = get_group(id_group=1)

# Récupérer les utilisateurs d'un groupe spécifique
users = get_group_user(1)

# Récupérer les tâches assignées à un groupe spécifique
tasks = get_group_task(1)
