Read Subtask¶

Ce module permet de récupérer des informations sur les sous-tâches en utilisant une API PHP avec des options de filtrage dynamiques.

Configuration de l’API¶

L’URL de l’API est définie comme suit :

url = "https://kicekifeqoa.alwaysdata.net/api.php"

Fonctions¶

get_data(table, columns='*', filter_column=None, filter_value=None, join_table=None, join_condition=None)¶

Récupère des données génériques d’une table avec des options de filtrage et de jointure.

Paramètres:

  • table (str) – Le nom de la table à interroger.

  • columns (str, optionnel) – Les colonnes à récupérer (par défaut “*” pour toutes les colonnes).

  • filter_column (str, optionnel) – La colonne utilisée pour le filtrage.

  • filter_value (str, optionnel) – La valeur utilisée pour le filtrage.

  • join_table (str, optionnel) – La table à joindre.

  • join_condition (str, optionnel) – La condition de jointure.

Renvoie:

Un dictionnaire JSON contenant les résultats ou un message d’erreur.

Type renvoyé:

dict ou str

Exemple d’utilisation :

data = get_data("Subtask", filter_column="id_subtask", filter_value=5)
print(data)

—

get_subtask(id_subtask=None, id_affected_task=None, name=None, end_date=None, checked=None)¶

Récupère des informations sur une ou plusieurs sous-tâches en fonction des filtres fournis.

Paramètres:

  • id_subtask (int, optionnel) – L’ID de la sous-tâche.

  • id_affected_task (int, optionnel) – L’ID de la tâche principale associée à la sous-tâche.

  • name (str, optionnel) – Le nom de la sous-tâche.

  • end_date (str, optionnel) – La date d’échéance de la sous-tâche au format “YYYY-MM-DD HH:mm:SS”.

  • checked (int, optionnel) – Statut de la sous-tâche (1 pour cochée, 0 pour non cochée).

Renvoie:

Une liste de dictionnaires contenant les informations des sous-tâches ou “no existing links” si aucune correspondance n’est trouvée.

Type renvoyé:

list ou str

Exemple d’utilisation :

# Récupérer une sous-tâche par ID
subtask = get_subtask(id_subtask=10)
print(subtask)

# Récupérer des sous-tâches avec plusieurs filtres
subtasks = get_subtask(name="Faire le ménage", checked=0)
print(subtasks)

Exemple de sortie réussie :

[
    {
        'id_subtask': 135,
        'id_affected_task': 5,
        'name': 'Nettoyer la cuisine',
        'end_date': '2024-06-01 15:00:00',
        'checked': 0
    }
]

Exemple de sortie en cas d’absence de correspondance :

'no existing links'
