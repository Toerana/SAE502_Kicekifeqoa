Delete User¶

Ce module permet de se connecter à une base de données MySQL et de supprimer des utilisateurs via une requête DELETE envoyée à une API PHP.

Configuration de l’API¶

L’URL de l’API est définie comme suit :

url = "https://kicekifeqoa.alwaysdata.net/api.php"

Configuration de la Connexion MySQL¶

config = {
    'user': '379269_admin',
    'password': 'Kicekifeqoa123*',
    'host': 'mysql-kicekifeqoa.alwaysdata.net',
    'database': 'kicekifeqoa_todolist',
}

Fonctions¶

close_connection_BDD(conn, cursor)¶

Ferme proprement la connexion à la base de données.

Paramètres:

  • conn (mysql.connector.connection.MySQLConnection) – L’objet de connexion à la base de données.

  • cursor (mysql.connector.cursor.CursorBase) – Le curseur de la base de données.

Exemple d’utilisation :

close_connection_BDD(conn, cursor)

—

delete_user(table, column, value)¶

Supprime un utilisateur de la base de données en envoyant une requête DELETE à l’API.

Paramètres:

  • table (str) – Le nom de la table où se trouve l’utilisateur.

  • column (str) – La colonne utilisée pour identifier l’utilisateur.

  • value (str) – La valeur de la colonne pour identifier l’enregistrement à supprimer.

Renvoie:

Aucun retour explicite, ferme la connexion après l’exécution.

Type renvoyé:

None

Exemple d’utilisation :

delete_user("Users", "email", "user@example.com")

Exemple de post_data envoyé à l’API :

{
    "table": "Users",
    "column": "email",
    "value": "user@example.com"
}

Exemple de sortie en cas de succès :

{"status": "success", "message": "User deleted successfully"}

En cas d’erreur :

{"status": "error", "message": "User not found"}
