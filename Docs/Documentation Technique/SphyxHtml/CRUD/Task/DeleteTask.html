Delete Task¶

Ce module permet d’interagir avec une base de données MySQL et une API PHP pour effectuer des opérations telles que la suppression de tâches.

Fonctions¶

close_connection_BDD(conn, cursor)¶

Ferme proprement la connexion à la base de données.

Paramètres:

  • conn (mysql.connector.connection.MySQLConnection) – L’objet de connexion à la base de données.

  • cursor (mysql.connector.cursor.CursorBase) – Le curseur de la base de données.

Exemple d’utilisation :

close_connection_BDD(conn, cursor)

—

delete_task(table, column, value)¶

Envoie une requête DELETE à l’API PHP pour supprimer une entrée spécifique dans une table.

Paramètres:

  • table (str) – Le nom de la table où se trouve la donnée à supprimer.

  • column (str) – Le nom de la colonne utilisée pour la condition de suppression.

  • value (str) – La valeur correspondant à l’entrée à supprimer.

Exemple d’utilisation :

delete_task("Tasks", "id", "5")

En cas de succès, la tâche spécifiée sera supprimée de la table.

—

Exemple Complet¶

# Connexion à la base de données
conn = connection.MySQLConnection(**config)
cursor = conn.cursor()

try:
    # Supprimer une tâche avec l'ID 5
    delete_task("Tasks", "id", "5")
    print("Tâche supprimée avec succès.")
finally:
    close_connection_BDD(conn, cursor)
